# ZhangNote (NeonMark Studio)

> AI 驱动的 Markdown 编辑器

## 产品定位
跨平台 Markdown 编辑器 + AI 知识管理工具

## 核心功能
| 类别 | 功能 |
|------|------|
| 编辑 | Markdown 编辑/预览、分屏模式、多标签、代码高亮 |
| AI | 对话、内容润色扩展、知识图谱、思维导图、测验生成 |
| 知识 | RAG 向量检索、文件夹管理、PDF/DOCX 导入 |
| 集成 | MCP 协议、语音转录、OCR、5 套主题 |

## 技术栈
- 前端：React 19 + TypeScript + Vite + Tailwind CSS v4
- 桌面端：Electron 33 + SQLite + LanceDB
- AI：Gemini / Ollama / OpenAI 兼容
