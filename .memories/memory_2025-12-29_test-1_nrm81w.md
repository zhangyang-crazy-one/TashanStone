# TashaStone 项目知识

## 项目概述
TashaStone 是一个 AI 驱动的 Markdown 编辑器，集成上下文工程技术。

## 核心功能
- Markdown 编辑与预览
- AI 对话助手
- 知识图谱可视化
- 永久记忆存储
- RAG 向量检索
- MCP 工具协议支持

## 技术栈
- 前端: React 19 + TypeScript + Vite
- 桌面端: Electron 33
- 数据库: SQLite + LanceDB
- AI: Gemini / Ollama / OpenAI
