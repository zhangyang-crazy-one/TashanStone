# 开发工作流指南

## 日常开发
1. 运行开发服务器: `npm run dev`
2. 执行测试: `npm test`
3. 构建应用: `npm run build`

## 代码风格
- 使用 TypeScript 严格模式
- 遵循 ESLint 规则
- 组件使用函数式编程

## 提交规范
- feat: 新功能
- fix: Bug 修复
- docs: 文档更新
- refactor: 重构
